# Bar-Clock

clock like bar on terminal

## Installation

After install [roswell](https://github.com/roswell/roswell), do following command

```
$ ros install myaosato/bar-clock
```

## Usage

```
$ bar-clock
2018-02-11 08:42:41
SECOND: *****************************************                    41 / 60
MINUTE: ******************************************                   42 / 60
HOUR  : ********************                                         8  / 24
DATE  : ***********************                                      11 / 28
MONTH : **********                                                   2  / 12
```
